My Portfolio

Honey Heist 'Bearacter' Generator

A Django app which 'rolls' a random character for the tabletop roleplaying game Honey Heist. Character class and stats are randomly generated as is character name assignment, but gender (or the absence thereof) is chosen by the user. This project was the completed as part of the Chicago Python User Group's (Chipy's) mentorship program.

Fake News

A Ruby on Rails app that pulls recent articles from the Guardian UK's api and allows the user to apply find and replace filters to make the content funnier. Version 1 includes the option to Swap, Add, and Redact article content.
